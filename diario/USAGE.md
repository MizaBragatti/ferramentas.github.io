# 🚀 Como Usar

## Desenvolvimento Local

```bash
# Instalar dependências
npm install

# Executar servidor local
npm start

# Acessar aplicação
http://localhost:3000
```

## Configuração

1. **Configure o Firebase:** Siga `FIREBASE_SETUP.md`
2. **Publique no GitHub:** Siga `GITHUB_DEPLOY.md`
